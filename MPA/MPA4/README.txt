Int to Words, Int to Roman Numerals, Roman Numerals to Int

1. Int to Words - Converts an input integer into words. e.g. "12", output: "TWELVE"
2. Int to Roman Numeral - Converts an input integer into Roman numeral. e.g. "123", output: "CXXIII"
3. Roman Numeral to Int - Converts an input Roman numeral into an integer e.g. "CXXIII", output: "123"

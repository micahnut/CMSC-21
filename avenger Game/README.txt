Avenger Game

The Game is all about defeating marvel's supervillains. You need to pick a team of 3 superheroes with different capabilities.

Use mouse and click start , help or quit.

Hover over the characters to choose your heroes. In game play, hover your mouse over the attack to attack the enemy.
Enemy's attack is automatic. Your goal is to defeat the enemy. 

This is a game project made using SFML. Use codeblocks to play this game. Run attempt1.7(withattackandlife).cpp using codeblocks to play.
